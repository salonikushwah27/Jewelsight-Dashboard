# 💎 JewelSight Sales Analytics Dashboard

JewelSight is an interactive **Power BI dashboard** that provides a comprehensive view of jewelry sales performance. It helps analyze business trends, monitor KPIs, and support data-driven decision-making through interactive visualizations.

## 📊 Dashboard KPIs
- 💰 Total Sales: **718.64M**
- 📈 Total Profit: **108.40M**
- 🛒 Total Orders: **5K**

## ✨ Key Features
- Brand-wise Sales Analysis
- Brand-wise Profit Comparison
- Sales by Product Category
- Sales Distribution by Metal Type
- Payment Mode Analysis
- Salesperson Performance Tracking
- Interactive Year, Month & City Filters
- Dynamic KPI Cards and Business Insights

## 🛠️ Tools & Technologies
- Power BI
- Power Query
- DAX
- Microsoft Excel
